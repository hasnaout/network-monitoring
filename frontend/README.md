# Frontend

Application React/Vite du projet de supervision reseau.

Scripts utiles :

```bash
npm install
npm run dev
npm run build
```

Depuis la racine du projet, vous pouvez aussi lancer :

```powershell
.\start-frontend.ps1
```

Variable d'environnement prise en charge :

- `VITE_API_URL` : URL de l'API FastAPI, par defaut `http://127.0.0.1:8000`
